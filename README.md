# Shader Editor and Viewer using ImGui
This is supposed to become a opengl (glsl) shader editor and viewer<br>
its using [ImGui](https://github.com/ocornut/imgui)

<br>

#### Folder structure
shader_editor/
- include/
    - glad
    - GLFW
    - imgui
    - KHR
- lib/
  - glfw3.dll
  - glfw3_mt.lib
  - glfw3dll.lib
- out/
  - glfw3.dll
  - roboto.ttf
- comp/
  - *output of `make comp_libs`*
